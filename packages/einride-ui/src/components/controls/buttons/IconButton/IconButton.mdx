## Accessibility

Since the button only includes an icon that is not discernible for screen reader users, a
descriptive `aria-label` must be provided.

```tsx
// bad
<IconButton aria-label="secondary-dialog-button" />

// good
<IconButton aria-label="Close dialog" />
```
