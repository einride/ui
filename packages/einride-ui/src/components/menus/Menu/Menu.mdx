Menu consists of these components:

- `<Menu>`: Contains all the parts of the menu.
- `<MenuTrigger>`: The button that toggles the dropdown menu.
- `<MenuContent>`: The content that pops out when the dropdown menu is open.
- `<MenuItem>`: The wrapper for each individual menu item.

```tsx
import { Menu, MenuTrigger, MenuContent, MenuItem } from "@einride/ui"
```
